About

This website indexes all man pages from the Arch official repositories. It was created for the man template on the Arch wiki, due to dissatisfaction with existing sites considered for this purpose. See this discussion for details.

The code powering this website can be found in the lahwaacz/archweb_manpages repository at GitHub.

On this website, you can find:

  • Listings of all man pages sorted alphabetically, by section or by language.
  • Listings of English man pages sorted alphabetically or by section (or similarly for any other language).
  • Listings of man pages in a particular section, e.g. all pages in section 1 or English pages in section 3.
  • Listings of man pages for a particular package, e.g. coreutils or core/acl.
  • And, of course, the content of each man page. There are {{ count_man_pages }} man pages from {{ count_pkgs_with_mans }} packages, another {{ count_pkgs_without_mans }} packages don't contain any indexable man pages.
